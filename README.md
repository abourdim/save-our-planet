# 🌍 Save Our Planet — v1.0

**An interactive trilingual web app for kids (6–12) about robots saving Earth in 2050.**

Inspired by Isaac Asimov's Three Laws of Robotics, this educational experience lets children explore 13 environmental zones, discover 39 specialized robots, and learn how technology can protect our planet.

---

## Features

- **13 Environmental Zones**: Oceans, Forests, Energy, Waste, Agriculture, Cities, Biodiversity, Climate, Construction, Education, Emergencies, Space, Health
- **39 Robot Agents**: Each zone has 3 unique robots with name, role, mission description, and power/speed/intelligence stats
- **100+ Chapter Outlines**: Every zone lists its full book chapters for future expansion
- **Trilingual Support**: French (FR), English (EN), Arabic (AR) with full RTL layout
- **Build Your Robot**: Kids name a robot, pick a zone, write a mission, choose a color, and generate a Robot ID Card
- **3 Laws of Robotics**: Kid-friendly explanation of Asimov's laws accessible from the hub
- **Custom SVG Illustrations**: 13 unique robot designs (submarine, drone, solar mech, crusher, harvester, city bot, tracker, weather bot, builder, teacher, rescue bot, satellite, medic) — all vector, animated, and color-matched per zone
- **Progress Tracker**: Zones marked as explored with a global progress bar

## Tech Stack

- **Single HTML file** — zero dependencies, works offline
- Pure vanilla JavaScript, CSS animations
- Inline SVG robot illustrations with CSS/SVG animations
- CSS Grid + Flexbox, responsive (mobile/tablet/desktop)
- CSS custom properties for theming
- Google Fonts: Orbitron, Exo 2, Noto Kufi Arabic

## How to Use

1. Open `save-our-planet.html` in any modern browser
2. Select your language (FR / EN / عربي)
3. Enter your agent name and start the mission
4. Explore zones, discover robots, read chapters
5. Build your own robot and generate an ID card

## Design

Dark mission-control aesthetic with neon glows, floating particles, and a HUD-style interface. Designed to appeal to kids who love action, tech, and mecha — while teaching environmental responsibility.

## Source Material

Based on the educational workshop document *"Il est temps de sauver la Terre"* by Association Workshop-DIY, which imagines a 2050 scenario where humans and robots cooperate to save the planet.
